NNAAMMEE
       openssl-version - print OpenSSL version information

SSYYNNOOPPSSIISS
       ooppeennssssll vveerrssiioonn [--hheellpp] [--aa] [--vv] [--bb] [--oo] [--ff] [--pp] [--dd] [--ee] [--mm]
       [--rr] [--cc]

DDEESSCCRRIIPPTTIIOONN
       This command is used to print out version information about OpenSSL.

OOPPTTIIOONNSS
       --hheellpp
           Print out a usage message.

       --aa  All information, this is the same as setting all the other flags.

       --vv  The current OpenSSL version.

       --bb  The date the current version of OpenSSL was built.

       --oo  Option information: various options set when the library was built.

       --ff  Compilation flags.

       --pp  Platform setting.

       --dd  OPENSSLDIR setting.

       --ee  ENGINESDIR settings.

       --mm  MODULESDIR settings.

       --rr  The random number generator source settings.

       --cc  The OpenSSL CPU settings info.

NNOOTTEESS
       The output of "openssl version -a" would typically be used when sending
       in a bug report.
